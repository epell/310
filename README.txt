First operation: Stack operations
	Using the numbers 1-4, select the operation desired. 
	If you are pushing, you will be prompted for a number.
	Any other operation will output the result to console
	Enter 0 to quit this operation

Second operation: arithmetic expression
	Enter an arithmetic expression, only numbers and + - * / are accepted.
	The result will be outputted to the console.
	Enter Q to quit this operation

Third operation: postfix notation
	Enter an arithmetic expression in postfix notation. Only single digits and + - * / are accepted
	The result will be outputted to the console.
	Enter Q to quit this operation and the program
